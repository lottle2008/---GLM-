## 1. 产品简介

### 1.1 什么是AI语音克隆与生成工具？

这是一个基于智谱AI开放平台的智能语音工具，集成三大核心功能：

| 功能 | 说明 |
|------|------|
| 📝 文生文 | 输入主题或关键词，AI自动生成适合配音的文本内容 |
| 🎙️ 语音克隆 | 上传参考音频，学习并模仿特定音色 |
| 🎧 语音合成 | 将文本转换为自然语音，支持多种音色选择 |

### 1.2 核心特点

- ✅ **双模式运行**：有无API Key都能使用
- ✅ **7+3音色组合**：7个官方音色 + 3个内置预设音色
- ✅ **中英双语支持**：支持中文和英文语音合成
- ✅ **简单易用**：浏览器操作，无需复杂配置
- ✅ **隐私安全**：所有数据本地处理

### 1.3 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10+ / macOS 10.14+ / Linux |
| 浏览器 | Chrome 80+ / Firefox 75+ / Safari 13+ / Edge 80+ |
| 网络 | 互联网连接（使用API功能时） |
| 其他 | 音频播放功能 |

## 2. 快速开始

### 2.1 安装与启动

#### Windows 系统

```batch
# 方法一：双击运行
双击 "run.bat" 文件

# 方法二：命令行运行
cd 项目目录
python app.py
```

#### macOS / Linux 系统

```bash
# 打开终端，运行
chmod +x run.sh
./run.sh

# 或直接运行
python3 app.py
```

### 2.2 访问应用

启动成功后，打开浏览器访问：

```
http://localhost:7860
```

### 2.3 首次使用配置

#### 配置API Key（推荐）

1. 点击右上角 **"API Key: 未配置"** 按钮
2. 输入您的智谱AI API Key
3. 点击 **"保存"** 按钮

> 💡 如何获取API Key？
> 访问 [智谱AI开放平台](https://open.bigmodel.cn/) 注册账号，在个人中心获取API Key。

#### 不配置API Key

您也可以不配置API Key，此时：
- 文生文功能不可用
- 语音克隆功能不可用
- 语音合成可使用**系统语音模式**

## 4. 功能使用

### 4.1 文生文功能

#### 使用场景
- 不知道该说什么，需要AI帮忙构思
- 需要生成专业、流畅的配音文案
- 希望获得多样化的内容建议

#### 操作步骤

1. **在文本框中输入内容**
   - 可以输入主题，如"介绍人工智能的发展历程"
   - 也可以输入具体要求，如"生成一段儿童故事的开头"

2. **选择生成方式**
   - ✨ **AI生成文本**：让AI根据您的输入生成内容
   - 📄 **使用输入文本**：直接使用您手动输入的文本

3. **等待生成完成**
   - 显示进度条
   - 生成完成后文本自动填入

4. **查看结果**
   - 生成的文本会显示在文本框中
   - 自动同步到语音合成步骤

#### 示例

| 输入 | 生成结果 |
|------|----------|
| "写一段关于春天的诗歌" | 春天的微风轻拂面颊，万物复苏百花香... |
| "生成产品介绍文案" | 我们的产品采用先进科技，为您带来前所未有的体验... |

### 4.2 语音克隆功能

#### 使用场景
- 希望使用自己或特定人的声音
- 需要保持音色一致性
- 希望声音更具个性化

#### 操作步骤

1. **准备参考音频**
   - 音频时长：10-60秒
   - 支持格式：wav, mp3, m4a, ogg, flac
   - 音质要求：清晰、无噪音、单人语音

2. **上传音频文件**
   - 点击上传区域
   - 或直接拖拽文件到上传区域

3. **输入音频文本**
   - 输入参考音频中朗读的内容
   - 文本内容需与音频完全匹配

4. **点击"开始克隆"**
   - 等待克隆完成
   - 克隆成功后自动播放预览

5. **保存音色**
   - 在弹窗中设置音色名称和备注
   - 点击保存将音色添加到音色库

#### 注意事项

| 要求 | 说明 |
|------|------|
| 音频时长 | 最短10秒，最长60秒 |
| 文本匹配 | 输入的文本必须与音频内容完全一致 |
| 音频质量 | 建议使用清晰的人声录音，避免背景音乐 |
| 网络要求 | 需要配置API Key才能使用此功能 |

### 4.3 语音合成功能

#### 使用场景
- 将文本转换为语音
- 选择不同的音色和风格
- 生成可下载的音频文件

3. **点击选择音色**

4. **生成音频**

## 5. 常见问题

### 5.1 API Key相关

#### Q: API Key是什么？在哪里获取？

API Key是智谱AI开放平台提供的数据访问密钥，用于身份验证。

获取步骤：
1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号并登录
3. 进入个人中心 → API Keys
4. 创建新密钥并复制

#### Q: 不配置API Key可以用哪些功能？

| 功能 | 无API Key | 有API Key |
|------|-----------|-----------|
| 文生文 | ❌ 不可用 | ✅ 可用 |
| 语音克隆 | ❌ 不可用 | ✅ 可用 |
| 智谱TTS | ❌ 不可用 | ✅ 可用 |
| 系统语音 | ✅ 可用 | ✅ 可用 |

#### Q: API Key安全吗？如何保护？

- API Key存储在浏览器本地Storage中
- 不会上传到任何服务器
- 建议不要在公共电脑上保存API Key

### 5.2 语音克隆相关

#### Q: 克隆的音色可以永久使用吗？

是的，克隆成功后音色会保存到本地数据库，可以永久使用。

#### Q: 上传的音频有什么要求？

| 要求 | 说明 |
|------|------|
| 时长 | 10-60秒 |
| 格式 | wav, mp3, m4a, ogg, flac |
| 内容 | 单人清晰人声 |
| 文本 | 需要提供对应的文本内容 |

#### Q: 克隆失败怎么办？

1. 检查音频格式是否正确
2. 确认音频时长在10-60秒范围内
3. 确保输入的文本与音频内容完全一致
4. 检查网络连接是否正常
5. 确认API Key配置正确且有效

#### Q: 克隆的音色可以删除吗？

可以，在音色管理页面可以删除自定义音色（预设音色不可删除）。

### 5.3 语音合成相关

#### Q: 生成的音频是什么格式？

默认生成 WAV 格式音频文件，采样率22050Hz。

#### Q: 音频文件保存在哪里？

音频文件保存在项目的 `outputs` 目录下，可通过界面在线播放或下载。

#### Q: 系统语音和智谱TTS有什么区别？

| 对比项 | 系统语音 | 智谱TTS |
|--------|----------|---------|
| 音质 | 基础 | 更加自然流畅 |
| 音色数量 | 7种固定 | 10种（含3个自定义） |
| 网络要求 | 不需要 | 需要 |
| 费用 | 免费 | 使用API配额 |

#### Q: 如何选择合适的音色？

| 场景 | 推荐音色 |
|------|----------|
| 儿童故事 | tongtong（彤彤）、luodo（罗多） |
| 新闻播报 | xiaochen（小陈） |
| 英文学习 | jam、chuichui |
| 产品介绍 | xiaochen（小陈） |
| 客服对话 | tongtong（彤彤） |

### 5.4 其他问题

#### Q: 页面打不开怎么办？

1. 确认服务已启动（终端无报错）
2. 检查端口7860是否被占用
3. 尝试清除浏览器缓存后刷新
4. 更换浏览器尝试

#### Q: 音频播放没有声音？

1. 检查电脑音量是否开启
2. 检查浏览器音频权限
3. 尝试使用耳机或外接音箱
4. 重启浏览器尝试

#### Q: 如何查看历史生成的音频？

音频文件保存在 `outputs` 目录下，可以直接访问文件。

## 6. 注意事项

### 6.1 数据安全

- 🔒 所有数据本地处理，不上传到服务器
- 🔒 API Key存储在浏览器本地
- 🔒 克隆的音色保存在本地数据库
- 🔒 生成的音频保存在本地目录

### 6.2 使用限制

| 项目 | 限制 |
|------|------|
| 单次文本长度 | 建议不超过1000字 |
| 音频文件大小 | 不超过10MB |
| 并发任务数 | 建议不超过3个 |
| 系统语音可用性 | 取决于操作系统 |

### 6.3 故障排查

| 问题 | 解决方案 |
|------|----------|
| 服务启动失败 | 检查端口7860是否被占用 |
| API调用失败 | 检查网络连接和API Key |
| 音频无法播放 | 检查浏览器音频权限 |
| 克隆失败 | 检查音频格式和文本匹配 |
| 页面加载慢 | 清除浏览器缓存或更换浏览器 |

### 6.4 最佳实践

1. **文生文**：使用具体、明确的指令效果更好
2. **语音克隆**：选择清晰、情感丰富的音频效果更好
3. **语音合成**：避免超长文本，分段生成效果更佳
4. **音色选择**：根据内容类型选择合适的音色
