## 1. 项目简介

### 功能概述

**AI 语音克隆与生成教育智能体 - 本地服务**是一款基于智谱AI开放平台的语音处理工具，提供三大核心功能：

| 功能 | 说明 |
|------|------|
| **文生文** | 利用AI生成文本内容 |
| **语音克隆** | 上传参考音频创建自定义音色 |
| **语音合成** | 将文本转换为语音 |

### 核心特点

- ✅ **双模式运行**：有API Key时启用全部高级功能，无API Key时使用本地免费方案
- ✅ **中英双语支持**：支持中文和英文语音合成
- ✅ **三步工作流**：文生文 → 语音克隆 → 语音合成
- ✅ **音色管理**：支持音色的搜索、备注、排序、分页
- ✅ **异步处理**：后台任务处理，不阻塞界面

## 3. 安装与运行

### Windows系统

#### 方式一：使用已打包的EXE（推荐）

1. 下载并解压 `ZhipuAudio.zip`
2. 双击运行 `ZhipuAudio.exe`
3. 自动打开浏览器访问 http://localhost:7860

#### 方式二：从源代码运行

```batch
# 1. 安装依赖
pip install flask requests

# 2. 运行服务
python app.py
```

### Mac系统

#### 方式一：使用DMG安装包（推荐）

1. 双击 `ZhipuAudio.dmg`
2. 将 `ZhipuAudio.app` 拖到 Applications 文件夹
3. 双击运行 `ZhipuAudio.app`
4. 自动打开浏览器访问 http://localhost:7860

> **注意**：首次运行可能需要在"系统设置" → "隐私与安全性"中允许运行。

#### 方式二：使用可执行文件

```bash
open dist/ZhipuAudio.app
```

### Docker部署

```bash
# 一键部署
./docker-run.sh

# 或手动部署
docker-compose up -d
```

## 4. 功能使用指南

### 步骤1：文生文

**功能**：生成或输入要合成语音的文本内容

**操作步骤**：

1. 在"步骤1：文生文"区域选择模式：
   - **AI生成**：输入生成指令，让AI自动生成文本
   - **手动输入**：直接输入或粘贴文本内容

2. 如果选择AI生成模式：
   - 在输入框中输入生成要求
   - 点击"生成文本"按钮
   - 等待AI生成完成

3. 生成的文本会显示在下方文本框中
4. 可以继续编辑文本内容

**示例指令**：
```
写一段关于人工智能的介绍，大约200字
```

---

### 步骤2：语音克隆

**功能**：上传参考音频创建自定义音色

**操作步骤**：

1. 点击"上传参考音频"按钮
2. 选择本地音频文件（支持：wav、mp3、m4a、ogg、flac）
3. **建议音频时长**：10-60秒
4. 在文本框中输入参考音频对应的文本内容
5. 点击"开始克隆"按钮
6. 等待克隆完成（约10-30秒）

**克隆成功后**：
- 会显示预览音频，点击可播放
- 会生成一个音色ID（voice_id）
- 点击"保存音色"可保存到本地数据库

> **注意**：此功能需要配置智谱API Key

---

### 步骤3：语音合成

**功能**：将文本转换为语音

**操作步骤**：

1. 在"步骤3：语音合成"区域选择模式：
   - **智谱TTS**：使用智谱AI语音合成（需要API Key）
   - **系统语音**：使用操作系统自带语音（免费）

2. 确认合成文本（自动从步骤1获取）

3. 选择音色：
   - 在音色列表中选择已保存的音色
   - 使用搜索框快速查找音色
   - 或选择系统预设音色

4. 点击"生成音频"按钮
5. 等待合成完成（约3-10秒）

**合成成功后**：
- 点击播放按钮预览音频
- 点击下载按钮保存音频文件

---

### 音色管理

**功能**：管理已保存的音色

**操作说明**：

| 功能 | 操作 |
|------|------|
| **搜索** | 在搜索框输入关键词，按Enter搜索 |
| **排序** | 点击表头（名称、创建时间、使用次数）排序 |
| **分页** | 使用底部翻页按钮浏览 |
| **编辑** | 点击编辑图标修改名称和备注 |
| **复制** | 点击复制图标复制音色ID |
| **删除** | 点击删除图标删除音色 |

**预设音色**：
- `klm` - 卡通1
- `mld` - 卡通2  
- `zhaoxue` - 赵老师

> **注意**：预设音色仅为示例，实际使用需要通过语音克隆创建

## 5. 配置说明

### API Key配置

**方法一：通过Web界面配置（推荐）**

1. 打开浏览器访问 http://localhost:7860
2. 点击右上角"API Key: 未配置"按钮
3. 在弹出的对话框中输入您的智谱API Key
4. 点击"确认"按钮

**方法二：通过环境变量配置**

```bash
# Linux/Mac
export ZHIPU_API_KEY=your_api_key_here

# Windows
set ZHIPU_API_KEY=your_api_key_here
```

> **获取API Key**：访问智谱AI开放平台 (https://open.bigmodel.cn/)

### 端口配置

默认端口为 **7860**，如需修改：

1. 打开 `app.py` 文件
2. 修改以下代码中的端口号：
```python
if __name__ == '__main__':
    port = 7860  # 修改此值
    app.run(host='0.0.0.0', port=port, debug=False)
```

## 6. 常见问题

### Q1：提示端口7860已被占用？

**解决方案**：

**Windows**：
```batch
netstat -ano | findstr ":7860"
taskkill /F /PID <进程ID>
```

**Mac/Linux**：
```bash
lsof -ti:7860 | xargs kill -9
```

### Q2：系统语音不工作？

**解决方案**：
- 系统语音依赖操作系统TTS引擎
- Windows：确保安装了语音包（设置 → 时间和语言 → 语音）
- Mac：确保系统语音已配置（系统设置 → 辅助功能 → 语音）
- 建议使用智谱TTS模式获得更好效果

### Q3：语音克隆失败？

**可能原因**：
- 参考音频质量太差
- 音频时长过短（建议10-60秒）
- API Key无效或余额不足
- 网络连接问题

### Q4：如何备份音色数据？

**解决方案**：
- 音色数据存储在 `voices.db` 文件中
- 定期备份此文件即可

### Q5：生成的音频保存在哪里？

**解决方案**：
- Windows：`ZhipuAudio.exe` 所在目录的 `outputs` 文件夹
- Mac：`ZhipuAudio.app` 所在目录的 `outputs` 文件夹
- Docker：`./outputs` 目录

### Q6：Windows提示"Windows保护你的电脑"？

**解决方案**：
1. 点击"更多信息"
2. 点击"仍然运行"

### Q7：Mac提示"无法打开，因为Apple无法检查其是否包含恶意软件"？

**解决方案**：
1. 打开"系统设置" → "隐私与安全性"
2. 在"安全性"部分找到提示信息
3. 点击"允许打开"
